Project developed in Unreal Engine focused on reproducing advanced gameplay mechanics and core systems.

Features

- Ekko's Ultimate (Chronobreak)  
  Rewinds the player's position, health, and state from a few seconds earlier using a custom timeline and state-tracking system.

- Lissandra's Ultimate (Frozen Tomb)  
  Can be self-cast to freeze and heal, or cast on an enemy to immobilize and deal area damage.

- Inventory System  
  Classic inventory with:
  - Item Pickup collect items from the world  
  - Item Drop drop items dynamically into the level  
  - Item Stack manage stackable resources (ammo, potions, etc.)  
  - UI Management view, use, and manage items through a clear interface
